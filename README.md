# 💳 Credit Card Fraud Detection - Data Analysis

## 📌 Project Overview
This project focuses on analyzing credit card transaction data to understand fraud patterns, data imbalance, and feature relationships. 

The goal is to explore the dataset using data analysis techniques and prepare it for future machine learning modeling.

---

## 📊 Problem Statement
Credit card fraud is a major financial issue worldwide. Fraudulent transactions are rare compared to legitimate ones, making the dataset highly imbalanced. 

This project performs exploratory data analysis (EDA) to understand:
- Class imbalance
- Feature distributions
- Correlation between variables
- Fraud vs Non-fraud transaction patterns

---

## 🛠 Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

---

## 📈 Exploratory Data Analysis Performed
- Data cleaning and inspection
- Checking missing values
- Class distribution analysis
- Correlation matrix visualization
- Feature distribution plots
- Fraud vs Non-fraud comparison

---

## 📂 Dataset
Dataset sourced from Kaggle:

https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud

Please download it from Kaggle and place `creditcard.csv` inside a `data/` folder before running the notebook.
